Problem Solver

I started at Century working as a Developer 1 on the Support team. Regularly had to troubleshoot backend process to find the root cause. An example I have is one day out PO process stopped working. We were no longer ingesting POs for any customer. The root cause was an exceptions procedure flagged the data not to be consumed at only the header level. SQL was trying to batch and it was failing on a NULL value because the detail level had no header level data. I skipped the data that was causing the error, correct the exceptions procedure to update all levels, and we processed all customer POs.
